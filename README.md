# NewsApp
Simple, lightweight app delivering news to your phone.
## Features
- Filter and browse news based on chosen criteria:
    - country,
    - category,
    - language,
    - keywords,
    - date,
    - domain.
- Easy access to the article source.
- Save interesting articles for later referencce.
